Skills Overview

HTML

HTML, or HyperText Markup Language, allows web users to create and structure sections, paragraphs, and links using elements, tags, and attributes. In my experience using HTML, it helps me develop webpages such as designing how a browser displays its elements like the texts, hyperlinks and media files. Also, I can easily navigate and insert links between related pages and websites as HTML is heavily used to embed hyperlinks. Lastly, HTML makes it possible to organize and format documents, similarly to Microsoft Word.

CSS

Cascading Style Sheet Language or also called CSS. In simple words, CSS helps me a lot to stylize elements written in the markup language or HTML on a different file. CSS is very helpful for us aspiring developers because it makes the actual HTML markup much cleaner and easier to maintain. In short, with CSS we don’t need to repeatedly describe how individual elements look. This saves time, shortens the code and makes it not as prone to errors.

Bootstrap

Bootstrap is a free and open-source web development framework. For me, this is the best tool to master because it is designed to ease the web development process of responsive, mobile-first websites by providing a collection of syntax for template designs. In simple words, it helps us aspiring web developers specially a beginner like me to build websites faster as they don’t need to worry about basic commands and functions. It consists of HTML, CSS, and JS-based scripts for various web design-related functions and components. Some of my favorite Bootstrap’s interface components are navigation bars, grid systems, image carousels, and buttons.

Sublime

Lastly is Sublime or Sublime Text. Some of the mentioned tools I used is quite familiar for me before I enter the Zuitt Bootcamp except for Sublime. It is simply the main tool or text editor we used for code, markup and others. Sublime Text's slick user interface along with its numerous extensions for syntax highlighting, source file finding and analyzing code metrics make the editor more accessible specially to new programmers like me. In simple words, it is a beginner-friendly source code editor that supports many programming and markup languages.
